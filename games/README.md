

# Games Directory

Add your single-file HTML games here!

## Guidelines

1. Each game should be self-contained in a single HTML file
2. Include a "Back to Portal" button linking to `../index.html`
3. Use responsive design for mobile compatibility
4. Add your game to `js/main.js` to appear in the portal

## Template

```html
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Your Game</title>
    <style>
        /* Your styles */
    </style>
</head>
<body>
    <a href="../index.html"><button>← Back</button></a>
    <!-- Your game content -->
    <script>
        // Your game logic
    </script>
</body>
</html>
```
